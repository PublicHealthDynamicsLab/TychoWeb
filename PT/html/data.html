Open Access to Global Health Data

1. You can explore and download data that have been used previously for published research. These datasets are highly curated and identical to the dataset used for some published paper.
2. You can explore and download disease data that have been standardized by Project Tycho, through our user interface.
3. You can programmatically access standardized disease data via our application programming interface (API) (forthcoming).

Data Format

Project Tycho CSV files contain counts of people diagnosed with conditions reported through public health surveillance. These CSV files include count values and attributes that specify the type of count or the population subgroup that was included. Project Tycho CSV files conform to the standard format described here.

Terms of Use

Project Tycho data can be used for any purpose as long as Project Tycho is attributed as the source, as specified by the Creative Commons Attribution License. For some datasets, additional attribution of the data contributor may be required, as indicated when downloading a dataset. Read more [here] about the Project Tycho license agreement. For questions about data use, please contact us by email tycho@phdl.pitt.edu.
